# 📚 PyLibrary - Library Management System

> Manage Books. Simplified.

---

## 📌 About the Project

PyLibrary is a simple terminal-based Library Management System
built in Python. It was created as part of the Capgemini Python
Training Mini Projects. The system allows you to add books, 
issue them to students, return them, and view all available books.

---

## ✨ Features

- 📖 Add new books with author and genre details
- 📋 View all available books in a formatted table
- 🎓 Issue books to students
- 🔄 Return issued books back to library
- 📚 Pre-loaded with 8 popular books
- ❌ Handles invalid choices gracefully
- 🗂️ Uses dictionary for organized data storage

## ▶️ How to Run


## 📋 How to Use

1. Run the program
2. Choose an option from the menu
3. Add / View / Issue / Return books
4. Press 5 to exit
